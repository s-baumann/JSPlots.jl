Images and Slides

This shows examples of filtered Pictures and Slides. Pictures can display static images, animated GIFs, or provide interactive filtering between multiple charts.

Regional Sales Analysis

Picture has three main uses:
1. Static Images: Display any image file (PNG, JPEG, SVG, etc.)
2. Animated GIFs: Embed animated visualizations
3. Filtered Viewing: Generate multiple charts and let users switch between them using filters.

This example shows filtered viewing - use the dropdowns above to explore sales data across different regions and products. The charts are generated with VegaLite. See here for more Picture examples

Pictures and Slides

Picture and Slides allow you to embed plots from Plots.jl, Makie.jl, and VegaLite.jl. Pictures can also display static images, animated GIFs, and can be filtered to show different charts based on user selections.
